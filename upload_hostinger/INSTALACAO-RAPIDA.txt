📌 RESUMO EM 7 PASSOS:

1️⃣  BANCO DE DADOS (phpMyAdmin)
   → Executar SQL: database/hostinger_migration.sql

2️⃣  FRONTEND (public_html/)
   → Upload: frontend/* para public_html/

3️⃣  BACKEND (api/)
   → Upload: backend/* para api/

4️⃣  CONFIGURAR .ENV (api/.env)
   → Renomear .env.example para .env
   → Preencher DB_PASSWORD e secrets

5️⃣  INSTALAR (SSH)
   cd api/
   npm install --production
   pm2 start index.js --name gracie-barra-api
   pm2 startup && pm2 save

6️⃣  TESTAR
   → https://seudominio.com.br/api/health
   → https://seudominio.com.br
   → Login: admin@graciebarra.com / admin123

7️⃣  ALTERAR SENHA DO ADMIN!
