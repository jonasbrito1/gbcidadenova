# Relatórios para Cliente - Gracie Barra Cidade Nova

Esta pasta contém todos os relatórios técnicos gerados para o cliente.

## Estrutura de Arquivos

### Relatórios de Segurança
- **Nome:** `relatorio-seguranca-YYYY-MM-DD.md`
- **Frequência:** Mensal ou após mudanças significativas de segurança
- **Conteúdo:** Análise de segurança, scan nmap, vulnerabilidades corrigidas

### Relatórios Semanais de Melhorias
- **Nome:** `relatorio-semanal-YYYY-MM-DD-DD.md`
- **Frequência:** Semanal
- **Conteúdo:** Melhorias implementadas, bugs corrigidos, atualizações gerais

## Padrão dos Relatórios

### Informações Obrigatórias
- Data do relatório
- Sistema/URL
- Responsável técnico
- Status geral

### Formatação
- Markdown (.md)
- Sem emojis
- Linguagem profissional e técnica
- Organizado por seções
- Checklist quando aplicável
